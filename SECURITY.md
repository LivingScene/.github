# Security Policy

We take the security of LivingScene seriously. Thank you for helping keep it and our users safe.

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Report privately via one of:

- **GitHub private vulnerability reporting** — the **Security** tab of the affected repo →
  *Report a vulnerability*.
- **Email** — [security@livingscene.ai](mailto:security@livingscene.ai).

Include a description, reproduction steps, the affected repo/version, and the impact. We aim to
acknowledge within **3 business days** and to keep you posted as we investigate and ship a fix.

## Supported versions

LivingScene is pre-1.0 and ships continuously; only the latest release of each surface
(hub.livingscene.ai, livingscene.ai, and the current desktop release) is supported. Please
reproduce against the latest version before reporting.
